# Currency Converter

## Project Overview
This project is a simple and responsive currency converter that allows users to convert an amount from one currency to another using real-time exchange rates from a public API.

## Features
- User input for amount
- Currency selection using dropdown menus
- Swap currency option
- Real-time conversion using API
- Loading and error handling
- Clean and responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Public Exchange Rate API

## How to Run the Project
1. Download or clone this repository
2. Open the index.html file in your browser
3. Enter amount and select currencies to convert

## Project Structure
- index.html
- style.css
- script.js

## Notes
An active internet connection is required to fetch real-time exchange rates.
